# 🌱 KAIROS – AI-Powered Precision Agriculture Platform

KAIROS is an intelligent precision agriculture platform that combines Artificial Intelligence, satellite remote sensing, IoT telemetry, and environmental analytics to help farmers monitor crop health, detect diseases early, and make data-driven farming decisions.

The platform integrates deep learning-based leaf disease detection, Sentinel-2 satellite NDVI analysis, weather intelligence, and IoT sensor data into a unified decision-support system for modern agriculture.

## ✨ Features

- 🤖 AI-powered leaf disease detection using deep learning
- 🛰️ Sentinel-2 satellite NDVI analysis
- 🌾 Farm registration with GIS polygon mapping
- 📍 Interactive Leaflet-based farm visualization
- 🌦️ Weather data integration
- 📡 IoT telemetry dashboard (temperature, humidity, soil moisture, NPK, pH, EC, etc.)
- 📊 Farm health scoring
- 📈 Analysis history and monitoring
- 💡 Intelligent agronomic recommendation engine
- 🔐 Secure JWT authentication and multi-farm management
